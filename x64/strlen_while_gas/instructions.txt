strlen

Task: Write a program to calculate the length of a string

Tips:
1) Use the "cmp" instruction before using the branch series of instructions
2) jmp: always
   jl:  branch less than (signed)
   jg:  branch greater than (signed)
   jle: branch less than or equal (signed)
   jge: branch greater than or equal (signed)
   jb:  branch less than (signed)
   ja:  branch greater than (signed)
   jbe: branch less than or equal (signed)
   jae: branch greater than or equal (signed)
   je:  branch if equal
   jne: branch if not equal
